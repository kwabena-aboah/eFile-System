eFile System (Inward and Outward Correspondence Register) application is used to manage documents such as memos, invitations and other related records of governmental institutions(ministries, assemblies etc).

The application contains the following navigation:
1.login.html (login page)
2 index.html (home page / dashboard)
3.inward.html (inward Correspondence page)
4.outward.html (outward Correspondence page)
5.profile.html (user profile page)
6.file_upload.html (Upload documents page)

The application should allow the user to perform login and user profile edit operations. The user should also be able to view a detailed report on the number of Correspondences the application has recorded. Import and Export options are available for the user too.
